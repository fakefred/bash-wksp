# Bash Workshop

Dependencies:

- LaTeX
- latexmk
- GNU Make (optional)

To compile, run `make`.

## Topics

Part 1 (WIP):

- Intro
    * Brief history
    * Unix philosophy
- Files & File Tree
    * `cat, cp, mv, rm`
    * `cd, pwd, ls, mkdir`
- CLI
    * Options and arguments
    * Reading man pages
    * Keyboard shortcuts
- Pipes
    * stdin, stdout
    * `>` and `>>`
    * `grep`

Part 2 (heavily WIP):

- Regex
    * Patterns
    * `grep -E`
    * `sed`
    * Capture groups
- Scripting
    * Variables
    * `if`
    * `for`

## Tarball

Before the workshop takes place, the directory `tarball/` is to be
distributed to every participant. It is intended to be in-workshop
exercise material.
