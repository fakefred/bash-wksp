# Bash Workshop

Dependencies:

- LaTeX
- latexmk
- GNU Make (optional)

To compile, run `make`.

## Topics

Part 1:

- Intro
    * Brief history
    * Unix philosophy
- Files & File Tree
    * `cat, cp, mv, rm`
    * `cd, pwd, ls, mkdir`
    * Paths, `.`, `..`, `*`
- CLI
    * Options and arguments
    * Reading man pages
    * Keyboard shortcuts
- Pipes
    * stdin, stdout
    * `>` and `>>`
    * Pipes
    * `grep`

Part 2:

- Regex
    * Patterns
    * `grep -E`
    * `sed`
    * Capturing groups
- Scripting
    * Variables
    * `if`
    * `for`

## Tarball

Before the workshop takes place, the directory `tarball/` is to be
distributed to every participant. It is intended to be in-workshop
exercise material.

## Cheatsheet

We will print the cheatsheet (source: `cheatsheet.tex`) in bulk and hand
out one copy per participant.
